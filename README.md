# ThatNightSky
> 내가 태어난 날 밤, 그 밤 하늘
  

`startchart.py` 파일에서 날짜, 장소를 입력한다.
```python
where = '부산광역시 가야동'
when = '1995-01-15 00:00'
```

다음 코드를 실행한다.
```shell
python starchart.py
```
  
내가 태어난 날 밤, 북쪽 밤하늘의 모습이 나온다.
![](./results/부산광역시_가야동__1995_01_15_00_00.png)
  
코드를 커밋한, 야근 날 서울 북쪽 밤하늘의 모습은 이렇다.
![](./results/서울시_강남구_신사역__2024_07_10_21_40.png)
